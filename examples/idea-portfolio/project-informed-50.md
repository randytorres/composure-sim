# Project-Informed 50

Built after reading recent project files for Forge Engine, REALM, SAMAEL,
Roviq Game Studio, PARAGON, and AGON/The Escape. These descriptions preserve
the concrete mechanics, architecture, and product surfaces already present in
the repos.

1. FORGE GAME - Prompt-to-playable native iOS game maker built on Forge's C++20 runtime, Swift gameplay/editor layer, direct Metal renderer, haptics/audio feedback stack, replay/debug hooks, and AI patch-review control plane.
2. PARAGON VANGUARD ALPHA - Mobile MMO superhero closed alpha with VANGUARD as the first playable faction, Public Trust, Court of Named People, identity exposure, rooftop arrival combat, civilian triage, and prestige drama as differentiation.
3. REALM VHAEGOR PROOF - Ship the Phase 1 dragon proof path: egg discovery, hatching ritual, dragonrider bond, Dragon POV short session, battlefield consequence, and Vhaegor the Storm-Bringer as the named dragon target.
4. SAMAEL BREACH ZERO - Facility, ship, and compound rehearsal surface where evidence becomes STRATA world package, physics graph, VECTOR routes, SPECTER RF, MURMUR adversary/civilian behavior, and signed AAR.
5. AGON THE FIRST WATCH - 100-man, 10-squad, 30-day Season 1 with $99 entry, Mission Card, Proof Feed, Locker Room, Captain orchestration, AGON Score, and Day-30 Pentathlon.
6. ROVIQ FORGE KIT - Shared adapter layer so PARAGON and REALM consume one Forge host bridge, one scene/runtime contract, and one gameplay/asset pipeline instead of drifting app-specific bridges.
7. SAMAEL CORE SDK - Public module SDK and conformance harness for physics modules, capability tokens, signed artifacts, CAL declarations, and third-party module trust classes.
8. PARAGON SAINT CALIBER FORGE SLICE - First real Forge/Metal gameplay proof using Saint Caliber rooftop arrival, power configuration, combat HUD, hit feedback, and app-to-engine bridge.
9. REALM COURT AND WHISPERS - Dense iOS strategy surface around Court of Named People, Loyalty/Honor/Ambition/Grudge, raven intel, planted deception, oath witnesses, succession threats, and hidden political state.
10. AGON CAPTAIN OS - Standalone operator/coach AI that runs recurring groups, posts daily check-ins, weekly recaps, mission previews, proof acknowledgments, quiet-member nudges, and leaderboard drops.
11. AGON COUNCIL API - Seven archetypal AAR voices with strict 5-part format, lineage-grounded citations, operator review flow, and the clean Captain-versus-Council separation.
12. FORGE PREMIUM FEEL LAB - Tooling and runtime assets for hit confirm, whiff, dodge, hurt, enemy windup, ambience, music state, haptic timing, thermal degradation, and content-backed feel diagnostics.
13. PARAGON IDENTITY INTEL - Superhero cover/exposure layer with secret identities, institutional surveillance, doxxing risk, faction intel, public trust cascades, and player-visible consequences.
